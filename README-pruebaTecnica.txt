# Archivos Agregados

## 1. Custom Slider
Sección donde el administrador de la tienda puede mostrar a los usuarios finales un slider completamente personalizable. Incluye:
- **2 archivos CSS**: Para la estilización del slider.
- **2 archivos JS**: Para la funcionalidad del slider.

## 2. Carrusel Tabs
Sección que permite al administrador de la tienda mostrar una lista de productos organizados por categorías junto con sus precios. Esta sección permite:
- Añadir múltiples categorías.
- Modificar fácilmente los precios y productos.

## 3. Countdown Timer
Sección que proporciona un contador regresivo para una oferta especial de producto. 

## Consideraciones
- Se utilizó data dummy extraída de la misma página para pruebas.
- Las funciones en los archivos JS están diseñadas para ser reutilizables.
- Se puede optimizar el código mediante la utilización de snippets.
